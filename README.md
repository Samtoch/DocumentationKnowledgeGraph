# DOCUMENT KNOWLEDGE GRAPH PROJECT
This is a .Net 10 project for backend services of document knowledge graph using semantic kernel with gpt-4, Neo4j, NLog, Automapper, Fluent Validator etc with Clean Architecture

# Prerequisites for running the project locally
- .NET 10 SDK
- Neo4j Graph Database
- Visual Studio 2022 or later / VS Code
- Postman or any API testing tool
- Docker (optional, for containerization)
- Swagger (for API documentation)
- Dapper (for database interactions)
- ASP.NET Core Identity (for user authentication and authorization)
- AutoMapper (for object mapping)
- FluentValidation (for input validation)
- NLog (for logging)
